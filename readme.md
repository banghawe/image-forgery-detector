# Image Forgery Detector

The program implement image forgery detector using block artifact grid(BAG) algorithm.


# Get Started
## Running the program
1. Install `pipenv`
2. Install packages using `pipenv`
    ```console
    pipenv install
    ```
3. Activate pipenv shell
    ```console
    pipenv shell
    ```
4. Run the program
    ```console
    python3 main.py
    ```
